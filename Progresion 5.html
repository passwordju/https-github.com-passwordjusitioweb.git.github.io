¿QUÉ ES HTML?

El HTML es un lenguaje informático conocido como lenguaje de marcas. La mayoría de las páginas web que ves en internet están construidas con sus etiquetas y códigos, entre muchas cosas más.² El código HTML le indica al explorador web lo que tiene que hacer con la información de la página mediante el marcado. El marcado muestra ciertas partes del texto de manera específica. Algunas partes del texto deben estar en negrita o en cursiva. Algunas partes del texto deben estar en tablas. Las etiquetas HTML le indican al explorador lo que tiene que hacer.

Los sitios web pueden ser como tiendas donde venden cosas, o como folletos digitales que te dan información sobre algo, como una escuela o una empresa. También pueden ser aplicaciones interactivas que juegas en línea. Para hacer una página web, empiezas con algo llamado HTML. Es como construir una casa: primero pones las paredes y el techo. HTML es como eso para una página web.

Una página web es como una página de un libro electrónico, donde puedes desplazarte hacia arriba y abajo para ver toda la información. Los desarrolladores usan archivos HTML para crear estas páginas. Un sitio web es como todo el libro electrónico, con muchas páginas juntas que puedes leer una después de otra.

HTML significa Lenguaje de Marcas de Hipertexto. Es como decirle a un navegador web cómo mostrar la información. Usas etiquetas en HTML para decirle al navegador qué partes del texto deben ser en negrita, en cursiva o en tablas. Por ejemplo, si quieres que algo esté en negrita, usas la etiqueta .
